# Marknotes
> A markdown editor

## Requirements
- [Marknotes api](https://github.com/gabrielmelogm/marknotes-api)

## Environment
- Node 20.x
- Next 13.x

## How to run
1. Install dependencies
```bash
pnpm install
```

2. Create <code>.env</code>
```bash
cp .env.example .env.local
```

3. Run app
```bash
pnpm dev
```
